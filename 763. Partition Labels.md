```python
class Solution:
    def partitionLabels(self, S: str) -> List[int]:
        last_index = {}
        for i, ch in enumerate(S):
            last_index[ch] = i
        res = []
        start = end = 0
        for i, ch in enumerate(S):
            end = max(end,last_index[ch])
            if i == end:
                res.append(end-start+1)
                start = end + 1
        return res
```
